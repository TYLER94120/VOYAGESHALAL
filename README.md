# Voyages Halal

Site web SEO-first pour le voyage halal — Next.js 14, TypeScript, Tailwind CSS.

## Stack

- **Framework** : Next.js 14 (App Router)
- **Langage** : TypeScript
- **Style** : Tailwind CSS
- **Déploiement** : Vercel

## Démarrage

```bash
npm install
npm run dev
```

## Structure

```
app/              # Pages Next.js (App Router)
components/       # Composants réutilisables
lib/              # Types, données, utilitaires SEO
```
